# python_todolist

<h1>KOR</h1>
오늘은 급하게 개인사정으로 아직 덜 완성된 todolist를 커밋합니다.<br><br>

아직 디자인면에서도 색바꿔야하는부분이 많구요.<br>
사용방법은 맨위에 "너의 오늘 할일이 뭐야?" 라는 string밑에 자신이 할일을 적습니다.
<br>그후 밑에 등록을 누르면 등록이 되구요. <br>
삭제를 하는 방법은 밑에 네모칸에 자신이 만약에 등록했을때 "개발" 이라는 것을 등록했다면 <br>
" [ 1 ] 개발 " 일텐데요 그러면 밑에 네모칸에 int형식으로 숫자를 쓰고 , Delete를 누르시면 됩니다.
